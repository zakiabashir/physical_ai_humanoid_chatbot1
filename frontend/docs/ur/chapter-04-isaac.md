# باب 4: این وڈیا آیزک

یہ باب این وڈیا آیزک کو کور کرتا ہے، جو کہ GPU ایکسیلریشن کا استعمال کرتے ہوئے فزکس اور رینڈرنگ کے لیے ایک جامع روبوٹکس سیمولیشن پلیٹ فارم ہے۔

## 4.1 این وڈیا آیزک پلیٹ فارم

| جزو | وضاحت |
|-----------|-------------|
| **آیزک سیم** | this one is based on NVIDIA Omniverse |
| **آیزک لیب** | روبوٹ لرننگ کے لیے ماڈیولر فریم ورک |
| **آیزک ROS** | NVIDIA ہاؤئیر کے لیے ROS 2 پیکیجز |
| **Jetson پلیٹ فارم** | روبوٹس کے لیے ایج کمپیوٹنگ |

## 4.2 آیزک سیم

آیزک سیم فراہم کرتا ہے:
- GPU-ایکسیلریٹڈ فزکس via PhysX
- فوٹوریالسٹک رینڈرنگ RTX کے ساتھ
- Domain رینڈمائزیشن sim-to-real ٹرانسفر کے لیے
- Multi-GPU سپورٹ پیرلل سیمولیشن کے لیے

## 4.7 آیزک سیم کے لیے مفت/کلاؤڈ متبادل

## خلاصہ

اس باب میں، آپ نے سیکھا:

- **آیزک سیم** GPU-ایکسیلریٹڈ فزکس سیمولیشن فراہم کرتا ہے
- **آیزک لیب** روبوٹ لرننگ کے لیے ایک فریم ورک ہے
- **ڈومین رینڈمائزیشن** sim-to-real ٹرانسفر کو قابل بناتا ہے
- **آیزک ROS** آیزک سیم کو ROS 2 کے ساتھ جوڑتا ہے
- **مفت متبادلات** جیسے PyBullet اور MuJoCo موجود ہیں
