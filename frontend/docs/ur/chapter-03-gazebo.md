# باب 3: گیزیبو اور ڈیجیٹل ٹوئنز

یہ باب گیزیبو کے بارے میں ہے، جو کہ روبوٹکس کے لیے ایک طاقتور فزکس پر مبنی سیمولیٹر ہے۔

## 3.1 ڈیجیٹل ٹوئن کیا ہے؟

ایک **ڈیجیٹل ٹوئن** کسی جسمانی سسٹم کی ورچوئل نمائندگی ہے۔

## 3.2 گیزیبو آرکائیکچر

| جزو | وضاحت |
|---------|-------------|
| **فزکس انجن** | رجیڈ باڈی ڈائنامکس کو سیمولیٹ کرتا ہے |
| **رینڈرنگ انجن** | حقیقی بصری effects پیدا کرتا ہے |
| **سنسر پلگ انز** | کیمرے، LIDAR، IMU وغیرہ کو سیمولیٹ کرتا ہے |
| **روبوٹ ماڈلز** | روبوٹ کی URDF/SDF تفصیلات |
| **ورلڈ فائلز** | ماحول، لائٹنگ، آبجیکٹس کو تعین کرتا ہے |

## 3.3 SDF (سیمولیشن تفصیل فارمیٹ)

## خلاصہ

اس باب میں، آپ نے سیکھا:

- **ڈیجیٹل ٹوئنز** جسمانی نظاموں کی ورچوئل کاپیاں ہیں
- **گیزیبو** فزکس پر مبنی سیمولیشن فراہم کرتا ہے
- **SDF** سیمولیشن کے لیے روبوٹ ماڈلز کو بیان کرتا ہے
- **فزکس انجنز** رجیڈ باڈی ڈائنامکس کو سیمولیٹ کرتے ہیں
- **سنسرز** پلگ انز کے ذریعے سیمولیٹ ہو سکتے ہیں
- **ورلڈ فائلز** سیمولیشن ماحول کو تعین کرتے ہیں
